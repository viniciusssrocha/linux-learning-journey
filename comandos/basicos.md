# Comandos Básicos Linux

Este documento reúne alguns dos comandos básicos do Linux que estou aprendendo durante a minha jornada de estudos.

---

## pwd

Mostra o diretório atual onde você está no sistema.

Exemplo:

```bash
pwd
```

Saída:

```
/home/vinicius
```

---

## ls

Lista arquivos e pastas do diretório atual.

Exemplo:

```bash
ls
```

Saída:

```
documentos downloads projetos
```

---

## ls -l

Lista arquivos com mais detalhes.

Exemplo:

```bash
ls -l
```

A saída mostra informações como:

* permissões
* dono
* tamanho
* data de modificação

---

## cd

Permite navegar entre diretórios.

Exemplo:

```bash
cd projetos
```

---

## cd ..

Volta uma pasta no diretório.

Exemplo:

```bash
cd ..
```
